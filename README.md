# 物理实验数据处理程序

## 功能说明

本程序用于物理实验数据处理，主要功能包括：

1. **双变量数据输入**：支持手动输入(x,y)数据对
2. **单变量数据输入**：支持输入单变量数据，自动搭配1,2,3...的横坐标
3. **统计分析**：计算平均值、方差等基本统计量
4. **最小二乘法拟合**：拟合直线并计算相关系数r
5. **图像生成**：生成拟合直线图像并保存，支持自定义坐标轴标签、单位和图表标题
6. **LaTex公式输出**：生成可直接用于论文的LaTex格式公式

## 安装依赖

本程序需要安装以下Python库：

```bash
pip install numpy matplotlib scipy
```

## 使用方法

1. 运行程序：
   ```bash
   python main.py
   ```

2. 按照菜单提示选择功能：
   - **选择1**：输入双变量数据(x,y)，每行输入一个x和y，用空格分隔，输入q结束
   - **选择2**：输入单变量数据，数据间使用空格隔开，输入q结束，横坐标自动设置为1,2,3...
   - **选择3**：显示统计结果，包括平均值、方差、斜率、截距、相关系数等
   - **选择4**：生成拟合图像，可以选择保存到文件或直接显示
   - **选择5**：输出LaTex格式的公式，可以选择保存到文件
   - **选择6**：退出程序

## 数据输入流程

### 双变量数据输入（选项1）

1. 输入数据对，每行一个x和y值：
   ```
   输入x y: 1.0 2.5
   输入x y: 2.0 4.8
   输入x y: 3.0 7.2
   输入x y: q
   ```

2. 输入完成后，程序会自动提示设置：
   - 横坐标变量名称（默认：X）
   - 横坐标单位（可留空）
   - 纵坐标变量名称（默认：Y）
   - 纵坐标单位（可留空）
   - 图表标题（可留空使用默认标题）

### 单变量数据输入（选项2）

1. 输入数据，数据间使用空格隔开：
   ```
   输入数据: 10.5 11.2 10.8 11.5 10.9
   输入数据: q
   ```

2. 输入完成后，程序会自动提示设置：
   - 横坐标变量名称（默认：序号）
   - 横坐标单位（可留空）
   - 纵坐标变量名称（默认：Y）
   - 纵坐标单位（可留空）
   - 图表标题（可留空使用默认标题）

## 注意事项

- 输入数据时请确保格式正确，双变量数据每行一个x和y，用空格分隔
- 单变量数据可以一次输入多个，用空格分隔
- 生成图像时，如果选择保存到文件，请确保路径存在，支持使用`~`表示用户主目录
- LaTex公式可以直接复制到LaTex文档中使用
- 程序使用scipy库的linregress函数进行最小二乘法拟合，结果准确可靠
- 每次操作后程序会自动重新显示功能菜单
