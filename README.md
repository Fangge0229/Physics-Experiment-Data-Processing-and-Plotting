# 物理实验数据处理程序

## 功能说明

本程序用于物理实验数据处理，主要功能包括：

1. **数据输入**：支持手动输入数据点
2. **统计分析**：计算平均值、方差等基本统计量
3. **最小二乘法拟合**：拟合直线并计算相关系数r
4. **图像生成**：生成拟合直线图像并保存
5. **LaTex公式输出**：生成可直接用于论文的LaTex格式公式

## 安装依赖

本程序需要安装以下Python库：

```bash
pip install numpy matplotlib scipy
```

## 使用方法

1. 运行程序：
   ```bash
   python 物理实验数据处理.py
   ```

2. 按照菜单提示选择功能：
   - 选择1：输入数据，每行输入一个x和y，用空格分隔，输入q结束
   - 选择2：显示统计结果，包括平均值、方差、斜率、截距、相关系数等
   - 选择3：生成拟合图像，可以选择保存到文件或直接显示
   - 选择4：输出LaTex格式的公式，可以选择保存到文件
   - 选择5：退出程序

## 注意事项

- 输入数据时请确保格式正确，每行一个x和y，用空格分隔
- 生成图像时，如果选择保存到文件，请确保路径存在
- LaTex公式可以直接复制到LaTex文档中使用
- 程序使用scipy库的linregress函数进行最小二乘法拟合，结果准确可靠
